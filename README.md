# mmorpg-lintingyang

# 文档介绍
## 职业
### 战士
- 拥有高攻击，高防御的职业
- 玩家每升一级，攻击力增加50，防御力增加50
- 具备【嘲讽】【回风拂柳】技能

### 牧师
- 具备治疗技能的职业
- 玩家每升一级，攻击力增加20，防御力20
- 具备【回春术】技能
### 法师
- 具体群体攻击技能的职业
- 玩家每升一级，攻击力增加30，防御力增加30
- 具备技能【剑荡八荒】
### 召唤师
- 具备召唤技能的职业
- 玩家每升一级，攻击力增加30，防御力增加30
- 具备技能【深渊召唤】

## 技能
- 回风拂柳：攻击类技能，单体攻击，对敌方造成150点伤害，冷却时间3秒
- 回春术：恢复类技能，瞬间恢复50点HP，冷却时间3秒，消耗50点MP
- 剑荡八荒：攻击类技能，攻击2人，对敌方造成50点伤害，冷却时间3秒，消耗50点MP
- 深渊召唤：召唤类技能，可以召唤出召唤师协助战斗，持续时间60秒，冷却时间3秒，消耗50点MP
- 嘲讽：嘲讽类技能，吸引敌方优先攻击自身，持续时间60秒，冷却时间3秒，消耗50点MP
- 连心刺：攻击类技能，攻击1人，对敌方造成56点伤害，冷却时间3秒
- 全垒打：攻击类技能，全体攻击，对敌方造成56点伤害
- 夺命冰：中毒效果，每隔3秒减少50点HP，持续时间60秒

## 场景
- 起始之地：新玩家诞生的地方，相邻场景【村子】
- 村子：相邻场景【起始之地】【野外】【黄巾前锋营】【竞技场】
- 野外：相邻场景【村子】
- 黄巾前锋营：副本场景，相邻场景【村子】
- 竞技场：玩家PK场景，相邻场景【村子】

## 商城
- 玩家可以使用币等货币在商城购买游戏道具
- 商品分为每日限购，每周限购，不限购
## 任务
- 到达某种条件后开启任务
- 完成任务可以获得一定的奖励道具

## 邮件
- 玩家之间可以发送邮件
- 可以附带奖励道具
- 系统可以通过邮件发放奖励
## 聊天
- 私聊：玩家之间可以进行一对一
- 全服：聊天信息全服玩家可见
## 背包
- 存放道具的地方，有容量上限
## 装备
- 穿戴装备，可以增加玩家防御力和攻击力
- 耐久度为零时装备失效，修复耐久度后重新生效
- 装备每提升一级增加6点攻击力，6点攻击力
## 公会
### 公会仓库
- 公会成员存储道具的地方
- 公会成员到达优秀会员有权限使用
### 公会职位
- 分为会长，副会长，优秀会员，普通会员新进会员
## 组队
- 玩家可相互组队一起攻打BOSS
## 竞技场
- 玩家之间可以相互PK
